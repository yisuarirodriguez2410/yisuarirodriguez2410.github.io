Trabajos

Investigaciones

Aprendimos cómo se construyen las clases en Java, sus miembros, modificadores de acceso y paquetes, con ejemplos prácticos para entender cómo organizar un programa.

  • Investigación 1

Hicimos un análisis FODA de NetBeans vs Eclipse, creamos un programa en Java para convertir temperaturas y diseñamos una red LAN de un colegio en Cisco Packet Tracer.

  • Investigación 2

Talleres Prácticos

Instalamos Windows Server 2022 e instalamos MySQL en nuestra máquina virtual y practicamos conexiones remotas entre ambas máquinas con Remote Desktop y SSH.

  • Taller práctico 1

Creamos un programa en Java para la empresa ficticia MariaPan, conectamos la base de datos y diseñamos su red en Cisco Packet Tracer. Al final generamos una factura simulada.

  • Taller práctico 2

Laboratorios

Instalamos y configuramos Java JDK y resolvimos problemas prácticos: un programa para simular datos de presentación y otro para crear facturas de una empresa.

  • Laboratorio 1

Hicimos programas que convertían la hora, calculaban el costo de conducir un carro, aplicaban descuentos a productos y calculaban sueldos según años de trabajo y aumentos.

  • Laboratorio 2

Ejercicios Prácticos

Hicimos programas para calcular la hipotenusa, mostrar nombres importantes, convertir monedas y calcular notas finales de estudiantes.

  • Ejercicio Práctico 1

Exámenes Parciales

Programamos en Java problemas de fechas, edades, años bisiestos y salarios, y diseñamos una red LAN con dos servidores en una oficina universitaria.

  • Parcial 1

Desarrollamos un sistema en Java para la empresa NIBARRA con menú, gestión de equipos, calendario de mantenimiento, control de usuarios, chat bot y simulación de red entre Chiriquí y Panamá.

  • Parcial 2
